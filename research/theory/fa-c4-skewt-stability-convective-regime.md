## 1. 问题:为什么现在会错

`diagnose_clouds`(clouds.py)用凝结物阈值/RH≥90% 找云层,只有**一套层状云几何**
(水平层 + 抛物线光路)。手册明确对流云(积云/积雨云,夏季常见)在几何模型上是
**另一种东西**:水平范围≈垂直高度,在水平压缩的天气图上近似**垂直线**(§1.2.3),
其火烧云只有垂直方向的时空分布,没有层状云那种"边界深入距离"的水平分布。把层状
云抛物线误用在对流云上,时长、范围、概率的推理链条全部错位。

## 2. 手册的物理(照录 + 最小重述)

### 2.1 气块抬升三阶段(§1.1.3 / §1.4.1)

把地面附近一团不饱和湿空气强行抬升:

1. **未饱和段**:气温沿**干绝热直减率** Γd ≈ 9.8 ℃/km 下降(海平面附近值,
   手册注明近地 3–4 km 适用);露点沿**等比湿线**下降,手册给值 **1.2 ℃/km**。
2. 气温 = 露点处气块饱和 —— 该高度即抬升凝结高度 **LCL**。由两条线的差速直接得
   手册内蕴的 LCL 公式:
   `z_LCL ≈ (T₀ − Td₀) / (9.8 − 1.2) km ≈ 116 m/K × (T₀ − Td₀)`
   (与文献 Espy 近似 z_LCL ≈ 125 m/K × (T−Td) 同量级;系数差来自手册取整的
   1.2 ℃/km。实现取手册系数,测试允许两者带宽。)
3. **饱和段**:沿**湿绝热直减率** Γm 下降;手册只给定性(|Γm| < |Γd|,水汽越多
   |Γm| 越小,水汽耗尽时 Γm→Γd),不给公式。实现采用标准伪绝热公式(AMS Glossary
   "moist-adiabatic lapse rate";亦见 Bolton 1980, MWR 108):
   `Γm = g·(1 + Lv·rs/(Rd·T)) / (cpd + Lv²·rs·ε/(Rd·T²))`
   其中 rs 为饱和混合比。该式在 T→很冷、rs→0 时自然退化为 Γd,与手册第 3 阶段
   描述一致(内建不变量)。

抬升全程的气块温度曲线 = 手册的**状态曲线**(state curve)。

### 2.2 条件不稳定与对流云高度(§1.4.1)

- 状态曲线在环境**温度层结曲线右侧**(T_parcel > T_env)→ 正浮力 → **不稳定**,
  可能出现对流;左侧 → 稳定,对流受抑制。
- **右偏区的大气厚度 = 对流活动发展的垂直高度 = 对流云的高度**。
- 手册明言:火烧云预报只需要**条件不稳定**这一种,不需要 CAPE/CIN 等更重的指标。

### 2.3 积云的定量斜温图判据(§2.2 积云)

- **云底以下直到地面**:温度直减率 ≈ 干绝热直减率(混合层特征,说明存在对流)。
- 云底以上状态曲线右偏层的厚度 → 积云等级:
  | 右偏厚度 | 云型 |
  |---|---|
  | 小几百米 | 淡积云(宽≫高,基本不构成对流云火烧云) |
  | 几百米 ~ 1.x km | 中积云(宽高相当) |
  | **≥ 2 km** | **浓积云(高≫宽;副高炎热天午后常发展为积雨云)** |
- 淡积云与对流性层积云层结相似,手册也承认有时分不清 —— 判别阈值天然带不确定带,
  实现必须把"接近阈值"暴露出来而不是硬切。

### 2.4 对流云火烧云的垂直线模型(§1.2.3(1),图 1.16/1.17)

坐标原点在对流云云底地面附近,t=0 为原点日落时刻,考虑晚霞(朝霞时间轴反向):

- 地影高度:`h_S(t) = l₀²/(2R) = (t·v)²/(2R)`(R 地球半径,v = −R·dφ/dt 为日落
  线速度,φ 太阳高度角 —— **v 即 FA-G4 已实现的终结线速度**)
- 被照亮高度:`h_L(t) = h_CT − h_S(t)`(h_CT 对流云云顶高度)
- **火烧云持续窗口:`0 ≤ t ≤ √(2R·h_CT)/v`**
- 常识性结论(手册原文):云顶越高,火烧云时间越长 —— 时长 ∝ √h_CT。

数量级自检:h_CT=10 km、R=6371 km、v≈18 km/min(手册伊春案例值)→
t_max = √(2·6371·10)/18 ≈ 357/18 ≈ **19.8 分钟**,与手册"高云晚霞 13~15 分钟"
同量级(对流云顶更高故略长),合理。

### 2.5 预报姿态(§4.1.2)

对流云火烧云属于**"不支持基于模式预报"的独立云况**:数值模式做不出有效的火烧云
估计,正确操作是 **(a) 识别出该云况并明示,(b) 交给实况临近预报**,同时照常给出
探空判云种、AOD 通透度等辅助判断。⇒ 实现层面:检出对流云况时**不是**把层状结论
换个几何再输出同等置信,而是输出"对流云况"标签 + §1.2.3 时长估计 + **降置信**,
条件指数可解释。

## 3. 假设与适用域

- 手册常数为准:Γd=9.8 ℃/km(近地 3–4 km 内当常数)、露点线 1.2 ℃/km;更高处
  仍沿用属于已知简化(状态曲线主要在低层比较,可接受)。
- 气块取**地面层起抬**(surface-based),不做 mixed-layer/most-unstable 变体
  (手册未要求,YAGNI)。
- 条件不稳定即止:不算 CAPE/CIN/EL 细节;对流云顶取右偏区顶(手册 §1.4.1 的
  "右偏厚度=对流云高度")。
- 判别边界(≥2 km 浓积云)带不确定带:±0.5 km 内输出"边缘"状态。
- 长三角经验参数的地域性:手册自称长三角适用;阈值进 config 可调。

## 4. 验证设计(先写失败测试)

Metamorphic/性质不变量(全离线合成廓线):

1. **LCL 线性**:z_LCL 随 (T₀−Td₀) 线性增长,系数 ≈116 m/K(带宽容 100–130)。
2. **湿绝热压干绝热**:任意 T、p 下 0 < Γm ≤ Γd;T→−60℃ 时 Γm→Γd(§1.1.3 第 3 阶段)。
3. **稳定度单调性**:环境(地面以上)变暖 → 右偏厚度单调不增;地面加热(T₀ 增大、
   露点不变)→ **对流云顶**单调不降(注:固定露点下加热抬高 LCL,硬盖之下右偏
   *厚度*可能合法缩小 —— 实现时发现的边界情形,不变量落在云顶而非厚度上)。
4. **全稳定廓线**(环境直减率 < 湿绝热)→ 右偏厚度 0 → regime=stratiform,
   现有行为零回归。
5. **教科书不稳定廓线**(干绝热混合层 + 条件不稳定中层)→ regime=convective,
   云顶 = 右偏区顶(构造已知解析解)。
6. **时长 ∝ √h_CT**:convective_duration(4·h)/convective_duration(h) == 2(同 v)。
7. **朝霞对称**:sunrise 与 sunset 的窗口时长相同(时间轴反向)。
8. 判别阈值扰动 ±ε 不翻转"远离阈值"的用例(阈值不确定带行为)。

对照:§2.4 的伊春数量级(v=18 km/min);`test_metamorphic_physics` 按 #57 验收扩充。
