# Historical CONUS case study

本目录保存项目早期使用 NOAA HRRR 和 Olympic Peninsula 案例研究 gate × modifier 架构的论文材料。

- `paper.tex` 是唯一正文源文件。
- `references.bib` 保存参考文献。
- `figures/*.py` 是图表源代码；PNG 保留用于离线编译。
- `paper.pdf` 是便于直接阅读的编译产物；修改正文后运行 `make` 同步更新。

这篇论文解释算法的物理起点，但其中的 CONUS 数据范围和历史实现不代表当前中国区产品架构。当前工作以 GitHub Project 和根目录 `README.md` 为准。
